# Cinemas Site

[This app](https://top-movies-bd.herokuapp.com/) shows ten most popular movies at the Moscow's cinemas

# How to local use


```#!bash

virtualenv -p python3 env
source env/bin/activate
pip install -r requirements.txt
python server.py
```
